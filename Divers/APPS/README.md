### Application multistream ###
Windows
  https://github.com/vzhd1701/gridplayer/releases/download/v0.5.4/GridPlayer-0.5.4-win64-portable.zip
or Via scoop:
$ scoop install gridplayer

Linux
  https://github.com/vzhd1701/gridplayer/releases/download/v0.5.4/GridPlayer-0.5.4-x86_64.AppImage
You may need to set execute permissions on AppImage file in order to run it:
$ chmod +x GridPlayer-0.5.4-x86_64.AppImage

Mac
  https://github.com/vzhd1701/gridplayer/releases/download/v0.5.4/GridPlayer.0.5.4.dmg
If you get "GridPlayer is damaged and can't be opened" error, run this command in the Terminal app:
$ sudo xattr -rd com.apple.quarantine /Applications/GridPlayer.app
